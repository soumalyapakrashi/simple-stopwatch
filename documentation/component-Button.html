Example
<Button
     text = 'Submit'
     displayChoice = 'text'
     buttonCategory = 'btn btn-orange'
     width = '100%'
     height = '100%'
     onClick = () => {}
/>
Parameters:
Name Type Description
text string The text which Button will display.
buttonCategory string The class which will get assigned to the Button.
onClick function Callback function which is executed when button is clicked.
iconUrl string The path in which the icon to be displayed is saved.
displayChoice string Choose whether a Text is to be displayed on the Button or an Icon.
altText string Alternative text for Icon if it is displayed.
width number The width of the Button (can be any unit accepted by CSS).
height number The height of the Button (can be any unit accepted by CSS).
